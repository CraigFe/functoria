module Alias: sig .. end

Alias

Alias allows to define virtual keys in terms of other keys at configuration time only.

type 'a t
The type for key alias.
val add : 'b Functoria_key.key ->
       ('a -> 'b option) -> 'a t -> 'a t
add k f a set a as an alias for the key k: setting a on the command-line will set k to f applied to a's value. If f returns None, no value is set.
val flag : Functoria_key.Arg.info -> bool t
flag is similar to Functoria_key.Arg.flag but defines configure-only command-line flag alias. Set stage to `Configure.
val opt : 'a Functoria_key.Arg.converter ->
       'a -> Functoria_key.Arg.info -> 'a t
opt is similar to Functoria_key.Arg.opt but defines configure-only optional command-line arguments. Set stage to `Configure.
