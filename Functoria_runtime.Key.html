Module Functoria_runtime.Key

module Key: sig .. end
Key defines values that can be set by runtime command-line arguments. This module is the runtime companion of Functoria_key.

Runtime keys


type 'a t
The type for runtime keys containing a value of type 'a.
val create : string -> 'a Functoria_runtime.Arg.t -> 'a t
create ~doc ~default conv create a new key.
val get : 'a t -> 'a
get k is the value of the key k. Use the default value if no command-line argument is provided.
val term : 'a t -> unit Cmdliner.Term.t
term k is the Cmdliner term whose evaluation sets ks' value to the parsed command-line argument.
