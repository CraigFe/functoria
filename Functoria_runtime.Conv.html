Module Functoria_runtime.Conv

module Conv: sig .. end
Converter defines Cmdliner-compatible argument converters. This is the runtime companion of Functoria.Dsl.Key.t

Argument Converters


type 'a t
The type for argument converters. It is a wrapper around Cmdliner.Arg.converter.
val flag : bool t
flag is a converter for an optional flag. The argument holds true if the flag is present on the command line and false otherwise.
val opt : 'a Cmdliner.Arg.converter -> 'a t
opt a wrap a Cmdliner.Arg.converter into a converter for optional arguments.
val int : int t
int is an integer converter for optional arguments.
val bool : bool t
bool is a boolean converter for optional arguments.
val string : string t
string is a string converter for an optional argument.
