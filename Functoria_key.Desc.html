Description of keys.

A complete description
val create : serializer:(Format.formatter -> 'a -> unit) ->
       converter:'a Cmdliner.Arg.converter ->
       description:string -> 'a t
To create a description we need:
serializer : A serialization function output the value as OCaml code.
converter : A cmdliner converter.
description : The identifier of the runtime description of the key. See Functoria_runtime.Desc.t.
