Module type Functoria.CUSTOM

module type CUSTOM = sig .. end
CUSTOM is the signature for custom DSL.
val prelude : string
Prelude printed at the beginning of main.ml.

It should put in scope:

  • a run function of type 'a t -> 'a
  • a return function of type 'a -> 'a t
  • a >>= operator of type 'a t -> ('a -> 'b t) -> 'b t

val name : string
Name of the custom DSL.
val version : string
Version of the custom DSL.
val driver_error : string -> string
driver_error s is the message given to the user when the the configurable s doesn't initialize correctly.
val argv : Functoria.Devices.argv Functoria_dsl.impl
argv is the device used to read the full list of command-line arguments.
val config : Functoria.Dsl.job Functoria.Dsl.impl list ->
       Functoria.Dsl.job Functoria.Dsl.impl
config jobs is the device configuration the custom DSL, given a list of configuration jobs.
